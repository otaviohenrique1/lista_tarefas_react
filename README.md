# lista_tarefas_react

## Scripts

### `npm start`

Roda o app no modo de desenvolvimento, abre no endereço [http://localhost:3000](http://localhost:3000).

### `npm test`

Roda os testes.

### `npm run build`

Roda o build do app para a produção na pasta `build`.

### `npm run eject`

Ejeta o app, operação irreversivel.
